# Memory Card Game

A responsive, highly interactive browser memory match game.

## Features

- **1-4 Players Mode**: Score tracking and turn management.
- **5 Levels**: 8 pairs (16 cards), 16 pairs (32 cards), 24 pairs (48 cards), 32 pairs (64 cards), or 40 pairs (80 cards).
- **Themes**:
  - Default: Country flags with country names (fetched via fast Flag CDN).
  - Custom uploads: Upload your own images, set custom names, and generate custom cards.
- **Web Audio FX**: Synthetic retro sound effects generated in-browser.
- **Responsive Layout**: Ink-on-Paper theme, designed with pure CSS.

## Development

Run local dev server:
```bash
npm run dev
```

Build:
```bash
npm run build
```
